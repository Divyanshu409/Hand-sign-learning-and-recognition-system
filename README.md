# 👋 Hand Sign Learning & Recognition System

An AI-based project for learning and recognizing hand signs using real-time video input.

## 🚀 Overview
This project focuses on making sign language learning interactive through real-time recognition and user-friendly features.

## ✨ Features
- Real-time hand sign recognition  
- Learning & practice modules  
- Interactive games  
- Chat functionality
